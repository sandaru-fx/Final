Follow these steps to set up the project locally.

### Prerequisites
* Node.js (v14 or higher)
* npm or yarn
* MongoDB (or your DB) installed locally

### 1. Clone the Repository
```bash
git clone [https://github.com/sandaru-fx/PawMate_Final.git](https://github.com/sandaru-fx/PawMate_Final.git)
cd PawMate_Final
```

2. Backend Setup
Navigate to the backend directory and install dependencies.

```Bash

cd backend
npm install
# Configure your .env file here (DB_URL, JWT_SECRET, etc.)
npm start
```

3. Frontend Setup
Open a new terminal, navigate to the frontend directory.

```Bash

cd frontend
npm install
npm start
```
